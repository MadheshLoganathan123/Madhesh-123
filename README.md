Title: Simple Login Page using HTML & CSS

This repository contains a simple and responsive login page built using HTML and CSS. The design is clean and user-friendly, featuring a login form with fields for a username and password. The page is styled with CSS to enhance its appearance and responsiveness.

Features:
✅ Clean and minimal design
✅ Responsive layout
✅ User-friendly login form
✅ Pure HTML and CSS (No JavaScript)
